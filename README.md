# Skyburst — Drone Fireworks Dodge

A web-based drone vs. fireworks dodging game built with HTML5 Canvas and Vanilla JavaScript.

## Play Online
You can play the game online at: **[https://ianworld.github.io/drone-fireworks/](https://ianworld.github.io/drone-fireworks/)**

## Controls
- **ARROWS / WASD**: Fly and steer your drone.
- **P**: Pause / Resume the game.
- **M**: Mute / Unmute sound effects and music.

## Features
- Survive 25 seconds per level and defeat the level's finale.
- Start with 5 lives (gain +1 life per level cleared).
- Dodge planes, balloons, UFOs, and lasers in later levels.
- Grab glowing power-ups including shields, speed boosts, catchers, and more.
- Color customization for your drone.
- Local high scores tracking.
